# cbjBlog

使用说明
    git clone https://github.com/binjie09/cbjBlog.git
    cd cbjBlog
    sh build.sh
    sh run.sh

## 作业

1. 在github上创建一个库  [✔]
2. 创建README.md        [✔]
3. 搭建一个博客，使用hugo[✔]
4. Dockerfile 把博客打包成一个镜像[✔]
5. build.sh 根据Dockerfile打包成镜像的shell脚本[✔]
6. run.sh 把项目跑起来的脚本[✔]
7. 日报
